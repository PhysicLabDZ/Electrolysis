# ⚡ PhysicLab — محاكاة التحليل الكهربائي

تطبيق ويب تفاعلي (PWA) لمحاكاة التحليل الكهربائي، موجّه لتلاميذ المرحلة المتوسطة.

## المميزات
- محاكاة حركة الشوارد والإلكترونات في الوقت الفعلي
- 6 محاليل مختلفة (NaCl, HCl, AgNO₃, SnCl₂, ZnCl₂, FeCl₂)
- ترسيب معدني على المهبط مع تأثير بصري واقعي
- تبويب الكشف عن الغاز (أزرق النيلة)
- تبويب استعمالات التحليل الكهربائي
- دعم كامل للغة العربية (RTL)
- يعمل بدون إنترنت (PWA)

## النشر على GitHub Pages

1. ارفع المحتوى على مستودع GitHub
2. اذهب إلى **Settings → Pages**
3. اختر **Branch: main** والمجلد **/ (root)**
4. انتظر دقيقة ثم افتح الرابط
